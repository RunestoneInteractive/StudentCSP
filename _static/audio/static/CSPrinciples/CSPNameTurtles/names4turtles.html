Assign a Name to a Turtle¶

Learning Objectives:

  • Use assignment to name objects like turtles.
  • Understand that using the right object at the right time, in order, is critical to success.

Names can be more than numbers and strings. They can also be turtles or “screens” (a space on the page where a turtle can draw). You can also call things like turtles and screens objects. Objects in programming are things that do the action in a program.

Bob Builds a House¶


Let’s combine the square and triangle code to create a house.





































  



 (Turtle_House) 








Let’s imagine going on to make another partial square along the slope, to create a “chimney.”



        

        
csp-5-5: Align the program pieces below to make the house and chimney figure above.  Draw the square first for the main part of the house, next draw the roof, and finally draw the chimney.


        

        
from turtle import *\nspace = Screen()\nbob = Turtle()
# Make a square\nbob.forward(100)\nbob.right(90)\nbob.forward(100)\nbob.right(90)\nbob.forward(100)\nbob.right(90)\nbob.forward(100)
# Position for roof\nbob.right(90)
# Make a roof\nbob.forward(100)\nbob.right(-120)\nbob.forward(100)\nbob.right(-120)\nbob.forward(100)\nbob.right(-120)
# Position for chimney\nbob.right(-60)\nbob.forward(40)\nbob.setheading(90)
# Draw chimney\nbob.pencolor("red")\nbob.forward(30)\nbob.right(90)\nbob.forward(30)\nbob.right(90)\nbob.forward(30)\nbob.right(90)

 Teacher Note: Body Syntonic¶


One of the things that Seymour Papert, the person who first used turtles in programming, most liked about the turtle is that students could test their programs using body syntonic reasoning.  This means that a turtle moves in the way that a student moves.  Students can go forward and turn right.  A student can pretend to be the turtle, doing exactly what the program says.  In this video, our “turtle” person, Barbara Ericson, doesn’t take 100 steps.  Instead, she goes forward 3 steps since her steps are so much larger than a turtle’s.
