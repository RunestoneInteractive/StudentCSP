Our first data set: Air pollution in the United States¶

This first data set that we’re going to explore is from The Guardian‘s data blog. They have a fascinating interactive map of all the cities in the world for which they have pollution data: http://www.theguardian.com/news/datablog/interactive/2014/may/08/exposure-air-pollution-city-map

The below data set is just the US data. There are three columns separated by colon (‘:’) characters:
  • The city and state, separated by a comma.
  • The annual mean amounts of particulate pollution that’s 10 micrometers in diameter, measured in terms of micrograms per cubic meter (ug/m3) of air.
  • The annual mean amounts of particulate pollution that’s 2.5 micrometers in diameter (even smaller), measured in the same micrograms per cubic meter of air.

According to the World Health Organization, 2.5 micrometer pollution is particularly deadly, because it more easily gets deep into our lungs. To give a sense of how dangerous that is, anywhere that the annual mean amount of just 5 µg/m3 (microgram per a cubic meter of air) was linked with a 13% increased risk of heart attacks.

Looking at all the data¶

That’s a lot of data. But we don’t actually know how much it is. We can’t use a for loop to process all of it. We need to use a while loop to keep going until we’re done.

We can process in by opening the file containing the data, then reading each line in. As long as the line is not empty, we keep going doing a split and processing the data. (This data is separated by a ”:” instead of a ”,”.)

In Python, we must open files before we can use them and close them when we are done with them. As you might expect, once a file is opened it becomes a Python object just like all other data. Table 1 shows the functions and methods that can be used to open and close files.

Method Name Use Explanation
open open(filename,'r') Open a file called filename and use it for reading. This will return a reference to a file object.
open open(filename,'w') Open a file called filename and use it for writing. This will also return a reference to a file object.
close filevariable.close() File use is complete.

What’s the largest pollution values?¶


One of the interesting questions to explore is where the pollution is the greatest.  The below program does that.  We read all the data in, then assume that the maxCity and the max25 (maximum PM2.5 value) is zero.  We then look for values that are greater than max25, and whenever we find one, we save that PM2.5 value (held in new25) and the city name.  We need the float function to convert the string value in values into a number we can compare.


What really took the most time in the last two programs is simply printing all the data.  We don’t need to make the code run longer here, so we can comment out the first two lines.

csp-17-2: Below is the program for computing the city with the largest PM10 pollution values.  Figure out the order here, then paste it into the program space below.


        

        
infile = open("uspollutiondata.txt","r")
lines = infile.readlines()
infile.close()

maxCity = ''
max10 = 0
for line in lines:
  values = line.split(":")
  new10 = float(values[1])
  if new10 > max10:
     maxCity = values[0]
     max10 = new10
print("Largest PM10 value is ",max10," in ",maxCity)
